## 🌱 เรื่องราวของโปรเจกต์นี้

โปรเจกต์นี้ไม่ได้เกิดจากห้อง co-working space ไม่ได้เกิดจากทีม developer มีเงินทุน และไม่ได้เกิดจากคนที่เรียนด้าน IT มาก่อน

มันเกิดจาก **ไรเดอร์คนหนึ่ง บนมือถือเครื่องเดียว ในคืนที่ชีวิตไม่เหลืออะไรให้ยึด**

วันหนึ่งผมต้องยกรถที่ทำมาหากินให้น้องสาวใช้ไปเรียน เพราะพ่อเครียดกับภาระค่าใช้จ่ายของน้อง และผมทนเห็นพ่อเป็นแบบนั้นไม่ได้ หลังจากนั้นพ่อตัดสินใจออกจากงานเพื่อหาเงินก้อน แต่กลับมาโดนคดีเก่าที่ตามมาจับ ต้องใช้เงินประกันตัว เลยต้องนำรถและข้าวของไปจำนำ

ผมไม่มีรถ ไม่มีรายได้ ไม่มีทุน

สิ่งที่เหลืออยู่คือ **มือถือเครื่องเก่า WiFi สาธารณะ และเวลา 24 ชั่วโมง**

ผมไม่เคยเขียนโค้ดมาก่อนในชีวิต แต่ผมรู้ว่าต้องหาทางออก เลยเริ่ม prompt AI ให้ช่วยเขียนโค้ด ทีละบรรทัด ทีละจุด แก้บัคทีละอัน รักษา context ข้ามหลาย account เพราะ token หมดกลางทาง ทำทุกอย่างใน Termux บนมือถือ RAM 3GB ที่แทบจะพัง

ใช้เวลาหลายคืน ใช้ 10 account เพื่อสร้างระบบนี้ขึ้นมา

ไม่ใช่เพื่อความสำเร็จ แต่เพราะ **ครอบครัวต้องการทางออก**

## ✨ Features

**ระบบ POS หลัก**
- จัดการสินค้าและราคาได้ไม่จำกัด
- รับชำระทั้งเงินสดและโอน
- ระบบสต็อกอัตโนมัติ ลดเมื่อขาย
- แจ้งเตือนเมื่อสินค้าใกล้หมด
- Export รายงานเป็น CSV ได้ทันที

**AI ประจำร้าน 🤖**
- คุยกับ AI เพื่อถามยอดขาย สต็อก แนวโน้ม
- **สั่งลงของผ่านแชทได้เลย** เช่น พิมพ์ว่า "น้ำส้ม 50 ขวดมาลงนะ" สต็อกขึ้นเองทันที
- ไม่ต้องกดอะไรเพิ่ม ใช้ภาษาบ้านๆ ได้เลย

**ระบบสมาชิก**
- เก็บประวัติลูกค้าประจำ
- ระบบสะสมแต้มและแลกรางวัล

**SaaS Multi-tenant**
- แต่ละร้านมี account และข้อมูลแยกกัน
- สมัครและเริ่มใช้งานได้ใน 3 นาที
- ระบบ admin อนุมัติและดูแลลูกค้า

## 🚀 Tech Stack

| ส่วน | เทคโนโลยี |
|------|-----------|
| Frontend | HTML / CSS / Vanilla JS (PWA) |
| Backend | Python + Flask |
| Database | JSON flat-file (ไม่ต้องพึ่ง DB server) |
| AI | Gemini API |
| Hosting | Railway |
| Auth | Session-based + SHA256 |

## 📦 วิธีติดตั้ง (Self-host)

```bash
git clone https://github.com/kerr77/Kasodd.git
cd Kasodd
pip install flask gunicorn
python app.py
```

เปิดเบราว์เซอร์ไปที่ `http://localhost:5000`

**Environment Variables ที่ต้องตั้ง:**
```
SECRET_KEY=your-secret-key
ADMIN_KEY=your-admin-key
```
